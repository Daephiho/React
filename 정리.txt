
보통 아래 도구를 이용해 리액트 애플리케이션을 만든다. 1~4챕터까지의 내용은 이걸 안 쓰고 만드는 법이였음. 

* create react app 도구 *

1. Nodejs 홈페이지에서 Nodejs를 다운받는다. -> 터미널에 node -v 를 입력해서 설치 확인을 한다. 
2. 터미널에 npx를 입력하고 문제가 없는지 확인한다.
3. 터미널에서 " npx create-react-app 원하는폴더명 " 입력해서 리액트 앱을 생성한다. 설치완료 후 happy hacking! 이 출력될 때 까지 기다린다.
4. 터미널에서 " code 입력한폴더명 " 을 입력해서 생성한 리액트 앱을 vscode에서 실행한다. 
5. 실행 후 터미널에서 npm start를 입력해 애플리케이션을 브라우저에서 실행한다. -> 실행이 잘 되는지 확인한다. 
5. 아주 많은 기본 파일들이 있다. 필요없는 건 삭제한 후 프로젝트를 시작하자.

터미널에서 npm i prop-types를 입력하여 prop-types를 설치하고 js파일 상단에서 임포트해준다.
